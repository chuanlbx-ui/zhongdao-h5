# GEMINI.md

## Project Overview

This is a React-based web application project scaffolded with Vite. It is configured to use TypeScript and Tailwind CSS for styling. The project also includes Zustand for state management and Axios for making HTTP requests. The application is a single-page application (SPA) and the development server is configured to proxy API requests to a backend server running on `http://localhost:3000`.

### Technologies

*   **Framework:** React
*   **Build Tool:** Vite
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **State Management:** Zustand
*   **HTTP Client:** Axios

## Building and Running

### Prerequisites

*   Node.js and npm (or yarn/pnpm)

### Development

To run the development server, use the following command:

```bash
npm run dev
```

The development server will be available at `http://localhost:5173`.

### Building

To build the project for production, use the following command:

```bash
npm run build
```

The production-ready files will be generated in the `dist` directory.

### Linting

To lint the project files, use the following command:

```bash
npm run lint
```
